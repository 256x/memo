# Tips & Tricks

## Markdown Toolbar

Use the toolbar at the bottom of the edit screen to quickly insert common Markdown syntax. Select text and tap a button to wrap it with tags.

## File Naming

Memos are saved with a timestamp filename (e.g., 20260324_143000.md).

When synced to GitHub, files are stored in the `pile/` folder.

## Search Tips

- Search covers the entire content
- Case-insensitive
- Press back button to clear search

## How GitHub Sync Works

- Active memos: `pile/filename.md`
- Trashed memos: `pile/.filename.md` (dot prefix)

You can edit files directly on GitHub or add new `.md` files to the `pile/` folder - they will be downloaded on next sync.

Permanent deletion removes files from both local and GitHub.

## Ideas for Use

- Quick notes and daily journaling
- Shopping lists with checkboxes
- Brainstorming with bullet points
- Technical notes with Markdown formatting
