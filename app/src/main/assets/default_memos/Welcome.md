# Welcome to Literal Memo

A simple, file-based Markdown memo app.

## Basic Operations

- **Create**: Tap the + button at bottom right
- **Edit**: Tap a memo to open → tap Edit button
- **Delete**: Swipe a memo to the left
- **Search**: Type in the search bar and press Enter

## GitHub Sync

Sync your memos to a GitHub repository for backup.

1. Create a Personal Access Token on GitHub (with repo scope)
2. Go to Settings → Connect GitHub
3. Enter your token and repository name (owner/repo)

## Trash

Deleted memos are moved to trash. Go to Settings → View Trash to restore or permanently delete them.
